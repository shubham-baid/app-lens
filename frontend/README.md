# Frontend

The frontend is a Next.js application that delivers AppLens user workflows from sign-in through graph exploration and AI-guided analysis.

## Responsibilities

- session-aware routing and authenticated navigation
- repository selection and scan execution flows
- graph visualization and interaction handling
- chat workflows for error analysis, what-if simulation, and natural-language questions
- typed API integration with backend contracts

## Key Areas

- app routes and layout
- reusable UI components
- API client and auth helpers
- feature-specific state and interactions
- frontend tests
