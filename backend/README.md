# Backend

The backend provides the AppLens API for authentication, repository scanning, dependency graph generation, and AI-assisted analysis.

## Responsibilities

- FastAPI app bootstrap, routing, and middleware
- database models, migrations, and async session management
- GitHub OAuth and authenticated user session handling
- scan orchestration across one or more repositories
- service and interaction graph extraction and retrieval
- analysis endpoints for error investigation, what-if impact, and architecture Q&A

## Key Areas

- app bootstrap and configuration
- database and migrations
- routes and request/response schemas
- service-layer orchestration
- detectors and graph builders
- backend tests
